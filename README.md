# Order Management System (Microservices)

A production-grade microservices architecture built with:
- Spring Boot
- Spring Cloud (Gateway, Eureka, Config Server)
- Kafka
- MySQL & MongoDB
- Redis
- Docker
- Resilience4j
- Prometheus + Grafana
- ELK Logging
- AWS EC2 Deployment

This project includes:
- Auth Service
- Order Service
- Inventory Service
- Payment Service
- Notification Service
- API Gateway
- Service Registry
- Config Server
